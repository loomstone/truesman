# Truesman — Voicebot Platform

Multi-tenant AI voice agent platform for restaurants.
Retell + Twilio + Supabase + FastAPI + Next.js.

## Architecture

- `apps/api` — FastAPI backend
- `apps/worker` — Background job worker
- `apps/dashboard` — Next.js admin dashboard
- `packages/pos` — POS adapter abstraction (Manual, Square, Clover, ...)
- `packages/db` — Supabase migrations
